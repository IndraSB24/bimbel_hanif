💡 Konsep Slicing PHP (Include)

Di PHP, kita tidak ingin menulis menu Sidebar dan Header berulang-ulang di setiap halaman. Oleh karena itu, desain di bawah ini nantinya akan kita potong (slice) menjadi 4 file:

  • sidebar.php : Berisi menu navigasi di sebelah kiri.
  • header.php : Berisi bagian atas (Profil user & sapaan).
  • footer.php : Berisi penutup tag HTML dan hak cipta di bawah.
  • index.php : File utama yang memanggil ketiga file di atas menggunakan include, dan hanya berisi konten inti di tengahnya saja.
