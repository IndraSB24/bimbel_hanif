Penjelasan Validasi Form dalam Integrasi HTML dan PHP

Apa itu Validasi Form?

Validasi form adalah proses memastikan data yang dimasukkan pengguna valid dan aman sebelum diproses. Ada dua jenis validasi:

  • Validasi Sisi Klien (Client-side): Dilakukan di browser menggunakan HTML dan JavaScript.
  • Validasi Sisi Server (Server-side): Dilakukan di server menggunakan PHP.

Validasi sisi klien cepat, tapi bisa di-bypass. Validasi sisi server lebih aman dan wajib untuk keamanan.

Contoh Validasi Sisi Klien (HTML)

Atribut Validasi HTML:

  • required: Input wajib diisi
  • minlength / maxlength: Panjang minimal/maksimal
  • min / max: Nilai minimal/maksimal (untuk number/date)
  • type="email": Validasi format email
  • type="url": Validasi format URL
  • pattern: Validasi dengan regex (contoh: pattern="[A-Za-z]{3,}")

Fungsi PHP untuk Validasi:

  • empty(): Cek apakah kosong
  • strlen(): Cek panjang string
  • filter_var(): Validasi email/URL
  • is_numeric(): Cek apakah angka
  • preg_match(): Validasi dengan regex

Untuk Pemula: Validasi seperti penjaga pintu. Sisi klien seperti penjaga pintu depan (cepat tapi bisa ditipu), sisi server seperti penjaga dalam rumah (lebih aman).
