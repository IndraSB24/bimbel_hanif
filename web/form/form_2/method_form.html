Penjelasan Method Form dalam Integrasi HTML dan PHP

Apa itu Atribut method?

Atribut method pada tag

menentukan cara data dikirim ke server. Ada dua method utama:

  • GET: Data dikirim melalui URL (query string). Terlihat di address bar browser.
  • POST: Data dikirim di body request. Lebih aman, tidak terlihat di URL.

Di PHP, data GET diakses via $_GET, data POST via $_POST.

Contoh Form dengan Method GET

Data akan muncul di URL: ?nama=...&email=...

Contoh Form dengan Method POST

Data tidak terlihat di URL, lebih aman untuk data sensitif.

Kapan Menggunakan GET vs POST?

  • Gunakan GET untuk: Pencarian, filter, bookmark-able pages (data bisa di-share via URL).
  • Gunakan POST untuk: Login, pendaftaran, upload file, data sensitif (password, dll).

Di PHP:

// Untuk GET
$nama = $_GET['nama'];

// Untuk POST
$nama = $_POST['nama'];

// Cek method
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    // Proses data POST
}

Untuk Pemula: GET seperti mengirim pesan terbuka (semua orang bisa lihat), POST seperti surat tertutup (hanya penerima yang tahu isi).
