Penjelasan Atribut name dalam Integrasi HTML dan PHP

Apa itu Atribut name?

Atribut name pada elemen form (input, textarea, select) memberikan nama unik untuk data yang dikirim. PHP menggunakan nama ini untuk mengakses data melalui $_POST atau $_GET.

Tanpa name, data tidak akan dikirim ke server!

Contoh Form dengan Name Attribute

Penjelasan Kode:

  • name="nama": Data input disimpan dengan key 'nama'. Di PHP: $_POST['nama']
  • name="email": Data input disimpan dengan key 'email'. Di PHP: $_POST['email']
  • name="pesan": Data textarea disimpan dengan key 'pesan'. Di PHP: $_POST['pesan']
  • Nama harus unik dalam satu form untuk menghindari konflik.
  • Untuk checkbox/radio dengan nama sama, gunakan array: name="hobi[]"

Untuk Pemula: Name seperti label pada kotak. PHP menggunakan label ini untuk menemukan isi kotak.
