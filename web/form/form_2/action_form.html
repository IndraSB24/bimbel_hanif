Penjelasan Action Form dalam Integrasi HTML dan PHP

Apa itu Atribut action?

Atribut action pada tag

menentukan URL atau file tujuan di mana data form akan dikirim. Biasanya, ini adalah file PHP yang akan memproses data.

Dalam integrasi HTML dan PHP, action menunjukkan ke file PHP yang akan menangkap data menggunakan $_POST atau $_GET.

Contoh Form dengan Action

Penjelasan Kode:

  • action="proses.php": Data form dikirim ke file proses.php.
  • Jika action kosong (action=""), data dikirim ke halaman yang sama.
  • Action bisa berupa URL lengkap, seperti action="https://example.com/proses.php".

Untuk Pemula: Action seperti alamat tujuan surat. Form mengirim "surat" (data) ke alamat tersebut (file PHP).
