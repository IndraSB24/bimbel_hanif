Materi Ajar HTML Forms - Bagian 2: Fungsi, Validasi, dan Layout

Apa yang Akan Kita Pelajari?

  • Fungsi Form: Cara form mengirim data ke server menggunakan PHP.
  • Validasi: Memastikan data yang dimasukkan valid (required, type email).
  • Layout: Menggunakan CSS Flexbox untuk tata letak yang rapi dan responsif.

Form ini akan mengirim data ke proses.php menggunakan method POST. PHP akan menangkap dan menampilkan data tersebut.

Form Pendaftaran

Penjelasan Teknis:

  • action="proses.php": URL tujuan pengiriman data (file PHP).
  • method="POST": Cara pengiriman data (aman untuk form).
  • required: Validasi sisi klien - browser akan cegah submit jika kosong.
  • type="email": Validasi format email otomatis.
  • name: Atribut penting untuk PHP mengidentifikasi input.
  • CSS Flexbox: Digunakan untuk layout responsif (lihat style.css).

Catatan: Validasi sisi klien (HTML) tidak cukup untuk keamanan. Di dunia nyata, tambahkan validasi sisi server di PHP.
