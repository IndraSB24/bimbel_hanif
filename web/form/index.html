Materi Ajar HTML Forms - Bagian 1: Elemen-elemen Dasar

Penjelasan Tag

Tag adalah wadah untuk elemen-elemen form. Atribut penting:

  • action: URL tujuan pengiriman data form (misalnya ke server atau halaman lain)
  • method: Cara pengiriman data. Nilai umum: "get" (data di URL) atau "post" (data di body request)
  • enctype: Encoding untuk data form (default: application/x-www-form-urlencoded)
  • target: Target window untuk response (_self, _blank, dll)
  • name: Nama form
  • autocomplete: Aktifkan/nonaktifkan autocomplete untuk semua input
  • novalidate: Skip validasi form
  • accept-charset: Charset yang diterima
