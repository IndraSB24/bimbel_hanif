Materi Ajar HTML Forms - Bagian 1: Elemen-elemen Dasar

Penjelasan Tag

Tag adalah wadah untuk elemen-elemen form. Atribut penting:

  • action: URL tujuan pengiriman data form (misalnya ke server atau halaman lain)
  • method: Cara pengiriman data. Nilai umum: "get" (data di URL) atau "post" (data di body request)

Contoh Lengkap Semua Elemen Form HTML

1. Input Text - Untuk teks biasa

2. Input Password - Untuk password (tersembunyi)

3. Input Email - Validasi email otomatis

4. Input Number - Hanya menerima angka

5. Input Tel - Untuk nomor telepon

6. Input URL - Untuk alamat website

7. Input Search - Untuk pencarian

8. Input Date - Untuk tanggal

9. Input Time - Untuk waktu

10. Input Datetime-local - Untuk tanggal dan waktu lokal

11. Input Month - Untuk bulan dan tahun

12. Input Week - Untuk minggu dan tahun

13. Input Color - Untuk memilih warna

14. Input Range - Untuk slider angka

15. Input Checkbox - Untuk pilihan multiple

16. Input Radio - Untuk pilihan tunggal

17. Input File - Untuk upload file

18. Textarea - Untuk teks panjang

19. Select - Dropdown untuk pilihan

20. Datalist - Autocomplete untuk input

21. Fieldset dan Legend - Grouping elemen form

Informasi Kontak

22. Button - Tombol untuk aksi

Catatan: Dalam bagian ini, kita fokus pada elemen HTML saja. Styling CSS sederhana ditambahkan untuk tampilan yang lebih baik, tapi logika pengiriman form belum dibahas (akan di bagian selanjutnya).
