Sesi 24: SQL - Create & Read Data

Belajar dasar manipulasi database dengan SQL.

1. Apa itu Create & Read?

CREATE

Proses membuat wadah (tabel) dan memasukkan data baru ke dalamnya. Menggunakan perintah CREATE TABLE dan INSERT INTO.

READ

Proses mengambil dan menampilkan data dari database. Menggunakan perintah sakti SELECT.

2. Persiapan: Skema 3 Tabel

Kita akan menggunakan contoh sistem perpustakaan sederhana dengan 3 tabel berikut:

Tabel: kategori

CREATE TABLE kategori (
  id_kategori INT PRIMARY KEY AUTO_INCREMENT,
  nama_kategori VARCHAR(50)
);

Tabel: penulis

CREATE TABLE penulis (
  id_penulis INT PRIMARY KEY AUTO_INCREMENT,
  nama_penulis VARCHAR(100)
);

Tabel: buku (Menghubungkan keduanya)

CREATE TABLE buku (
  id_buku INT PRIMARY KEY AUTO_INCREMENT,
  judul VARCHAR(200),
  id_penulis INT,
  id_kategori INT
);

3. Memasukkan Data (CREATE)

Setelah tabel dibuat, kita isi dengan data menggunakan INSERT INTO.

-- Isi Tabel Kategori
INSERT INTO kategori (nama_kategori) VALUES ('Fiksi'), ('Teknologi');

-- Isi Tabel Penulis
INSERT INTO penulis (nama_penulis) VALUES ('Tere Liye'), ('Elon Musk');

-- Isi Tabel Buku
INSERT INTO buku (judul, id_penulis, id_kategori) VALUES
('Hujan', 1, 1),
('Pulang', 1, 1),
('SpaceX Mission', 2, 2);

4. Membaca Data (READ)

Cara mengambil data dari satu tabel atau lebih.

A. Mengambil Semua Kolom

SELECT * FROM buku;

B. Mengambil Kolom Tertentu

SELECT judul FROM buku;

C. Membaca Data dengan Filter (WHERE)

"Tampilkan buku yang termasuk kategori Fiksi (ID 1)"

SELECT * FROM buku WHERE id_kategori = 1;
