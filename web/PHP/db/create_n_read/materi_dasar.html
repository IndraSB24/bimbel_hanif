Sesi 24: SQL - Create & Read Data

Belajar dasar manipulasi database dengan SQL.

1. Apa itu Create & Read?

CREATE

Proses membuat wadah (tabel) dan memasukkan data baru ke dalamnya. Menggunakan perintah CREATE TABLE dan INSERT INTO.

READ

Proses mengambil dan menampilkan data dari database. Menggunakan perintah sakti SELECT.

2. Persiapan: Skema Tabel Master

Pastikan kamu sudah membuat tabel-tabel utama ini sebelum mengisi data:

Tabel: kategori, penulis, dan buku

-- Contoh salah satu tabel
CREATE TABLE buku (
  id_buku INT PRIMARY KEY AUTO_INCREMENT,
  judul VARCHAR(200),
  id_penulis INT,
  id_kategori INT
);

3. Memasukkan Data (CREATE)

Gunakan INSERT INTO untuk memasukkan minimal 10 data agar database bervariasi.

-- Masukkan banyak data sekaligus
INSERT INTO kategori (nama_kategori)
VALUES ('Fiksi'), ('Sains'), ('Sejarah'), ('Biografi'), ('Komik');

4. Membaca Data (READ) Dasar

-- Ambil semua data
SELECT * FROM buku;

-- Ambil kolom tertentu saja
SELECT judul, id_penulis FROM buku;

5. Operator dalam Klausa WHERE

Operator digunakan untuk memfilter data agar lebih spesifik sesuai kebutuhan laporan.

Operator Deskripsi Contoh Penggunaan
= Sama dengan WHERE id_kategori = 1
!= atau <> Tidak sama dengan WHERE nama_peminjam != 'Budi'
> dan < Besar dari / Kecil dari WHERE tgl_kembali < '2025-11-10'
LIKE Pencarian pola teks (paling sering pakai %) WHERE nama LIKE 'Bu%' (Cari yang awalan Bu)
BETWEEN Di antara rentang nilai WHERE tgl_pinjam BETWEEN '2025-01-01' AND '2025-01-31'
IN Cocok dengan salah satu nilai di list WHERE id_kategori IN (1, 3, 5)

Contoh Query Filter Lanjutan:

-- Mencari peminjaman yang "Jatuh Tempo" (sebelum hari ini)
SELECT * FROM peminjaman WHERE tgl_kembali < '2025-11-10';

-- Mencari nama peminjam yang mengandung huruf "Siti"
SELECT * FROM peminjaman WHERE nama_peminjam LIKE '%Siti%';
