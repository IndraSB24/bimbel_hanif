Sesi 24: SQL - Create, Read & Analytics

Belajar manipulasi dan analisis data dengan SQL.

1. Apa itu Create & Read?

CREATE

Proses membuat wadah (tabel) dan memasukkan data baru ke dalamnya. Menggunakan perintah CREATE TABLE dan INSERT INTO.

READ

Proses mengambil dan menampilkan data dari database. Menggunakan perintah sakti SELECT.

2. Persiapan: Skema Tabel Master

Pastikan kamu sudah membuat tabel-tabel utama ini sebelum mengisi data:

Contoh Tabel Buku

CREATE TABLE buku (
  id_buku INT PRIMARY KEY AUTO_INCREMENT,
  judul VARCHAR(200),
  id_penulis INT,
  id_kategori INT
);

3. Memasukkan Data (CREATE)

Gunakan INSERT INTO untuk memasukkan data sekaligus. Sangat disarankan memasukkan banyak data agar laporan lebih bervariasi.

-- Masukkan data kategori sekaligus
INSERT INTO kategori (nama_kategori)
VALUES ('Fiksi'), ('Sains'), ('Sejarah'), ('Biografi'), ('Komik');

4. Membaca Data Dasar

-- Ambil semua kolom
SELECT * FROM buku;

-- Ambil kolom judul dan penulis saja
SELECT judul, id_penulis FROM buku;

5. Filter Data dengan Klausa WHERE

Digunakan untuk menyaring data agar yang muncul hanya yang kita butuhkan saja.

Operator Deskripsi Contoh
= Sama dengan WHERE id_kategori = 1
!= Tidak sama dengan WHERE nama != 'Budi'
> , < Besar / Kecil dari WHERE tgl < '2025-11-10'
LIKE Pola Teks (%) WHERE nama LIKE 'Bu%'
BETWEEN Antara rentang WHERE id BETWEEN 1 AND 10

6. Sorting & Analisis Data (Reporting)

Agar laporan data lebih rapi dan profesional, kita perlu mengurutkan atau menghitung datanya.

A. Pengurutan (ORDER BY)

Mengurutkan data dari terkecil (ASC) atau terbesar (DESC).

-- Urutkan berdasarkan judul dari A ke Z
SELECT * FROM buku ORDER BY judul ASC;

-- Urutkan transaksi terbaru ke terlama
SELECT * FROM peminjaman ORDER BY tgl_pinjam DESC;

B. Menghapus Duplikat (DISTINCT)

Menampilkan nilai-nilai yang unik saja (tidak ada yang dobel).

-- Tampilkan daftar nama peminjam (tanpa pengulangan nama)
SELECT DISTINCT nama_peminjam FROM peminjaman;

C. Fungsi Agregasi (Penghitungan)

Perintah untuk melakukan perhitungan otomatis pada kolom.

COUNT()
Hitung Baris
SUM()
Jumlahkan Total
AVG()
Rata-rata
MAX/MIN
Nilai Tertinggi
-- Hitung total buku yang dipinjam
SELECT COUNT(*) FROM peminjaman;

-- Cari harga buku paling mahal
SELECT MAX(harga) FROM buku;
