Soal 1: Tabel Guru Sederhana

Buat tabel bernama guru dengan kolom:
  • id: Angka, Primary Key, Auto Increment.
  • nama_lengkap: Teks (max 100), Wajib diisi.
  • mata_pelajaran: Teks (max 50).

Soal 2: Tabel Perpustakaan (Constraints)

Buat tabel buku dengan aturan ketat:
  • kode_buku: INT, Primary Key.
  • judul: VARCHAR(200), Wajib diisi.
  • isbn: VARCHAR(20), Harus Unik (UNIQUE).
  • tahun_terbit: INT (4 digit).
  • stok: INT, Default 0.

Soal 3: Tabel Produk dengan ENUM

Buat tabel produk:
  • id: INT, Primary Key, Auto Increment.
  • nama_barang: VARCHAR(100).
  • kondisi: Pilihan (ENUM) hanya boleh 'Baru' atau 'Bekas'.
  • harga: DECIMAL(10, 2) untuk uang presisi.
