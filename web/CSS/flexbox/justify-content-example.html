Ringkasan Justify Content Values

Value Description Use Case
flex-start Items align to the start of the container Default alignment, left-aligned content
flex-end Items align to the end of the container Right-aligned content, footer elements
center Items centered in the container Centered navigation, modal dialogs
space-between Equal space between items, first/last at edges Navigation menus, button groups
space-around Equal space around each item Card layouts, evenly spaced elements
space-evenly Equal space between items and edges Perfectly balanced layouts
