Penjelasan:

Layout ini menggunakan flex-wrap: wrap dan justify-content: space-between untuk mendistribusikan kotak secara merata. Container dibatasi lebar maksimal 1000px untuk memastikan maksimal 5 kotak per baris.

Setiap kotak memiliki lebar berbeda (70px - 200px) tetapi tinggi tetap (100px). Total ada 15 kotak yang tersusun dalam 3 baris.

Coba resize browser untuk melihat bagaimana kotak berpindah baris dan penyebaran berubah!
