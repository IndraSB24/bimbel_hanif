Demo CSS Flexbox: Basic Colored Boxes

Contoh sederhana Flexbox dengan kotak-kotak berwarna yang menunjukkan distribusi ruang. Mirip dengan demo klasik yang sering digunakan untuk menjelaskan konsep Flexbox.

Penjelasan:

Kotak-kotak di atas menggunakan display: flex pada container dengan justify-content: space-around untuk mendistribusikan ruang secara merata di sekitar setiap kotak.

Setiap kotak memiliki lebar tetap (100px) dan tinggi (100px) dengan warna latar berbeda untuk memudahkan identifikasi.
