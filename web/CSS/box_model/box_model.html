Penjelasan Box Model

  • Margin (Batas Luar)

    Area transparan di luar border. Fungsi utamanya adalah untuk memberi jarak dan mendorong elemen lain menjauh dari kotak ini. Margin tidak memiliki warna background. 

    /* Contoh kode CSS untuk Margin */
.elemen-saya {
  margin: 20px; /* 20px di semua sisi */
  margin-top: 10px; /* 10px hanya di atas */
  margin-left: 5px; /* 5px hanya di kiri */
}
  • Border (Garis Tepi)

    Garis yang "membungkus" padding dan content. Border memiliki ketebalan (border-width), gaya (border-style: solid, dashed, dll.), dan warna (border-color). 

    /* Contoh kode CSS untuk Border */
.elemen-saya {
  border: 2px solid #333; /* 2px, solid, warna gelap */
  border-bottom: 4px dashed red; /* 4px, putus-putus, merah */
}
  • Padding (Bantalan)

    Area transparan di dalam border, yang mengelilingi content. Padding memberi "ruang napas" untuk isi di dalamnya dan akan mengambil warna dari background-color elemen. 

    /* Contoh kode CSS untuk Padding */
.elemen-saya {
  padding: 15px; /* 15px di semua sisi */
  padding: 10px 20px; /* 10px atas/bawah, 20px kiri/kanan */
}
  • Content (Isi)

    Bagian pusat kotak tempat konten Anda (teks, gambar, video) ditampilkan. Ukurannya diatur oleh properti width dan height. 

    /* Contoh kode CSS untuk Content */
.elemen-saya {
  width: 300px;
  height: 150px;
  box-sizing: border-box; /* Sangat direkomendasikan! */
}
