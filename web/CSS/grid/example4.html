Demo CSS Grid: Column Spans

CSS Grid memungkinkan item untuk menjangkau beberapa kolom atau baris menggunakan grid-column dan grid-row. Ini berguna untuk membuat layout yang lebih kompleks.

Penjelasan Column Spans

grid-column: span n membuat item menjangkau n kolom.

grid-column: start / end menentukan garis kolom awal dan akhir.

Contoh penggunaan:

  • grid-column: span 2; - Jangkau 2 kolom
  • grid-column: 1 / 4; - Dari kolom 1 sampai 4
  • grid-column: 2 / -1; - Dari kolom 2 sampai kolom terakhir

Catatan: Angka negatif menghitung dari akhir grid.
