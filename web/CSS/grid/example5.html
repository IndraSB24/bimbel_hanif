Penjelasan Auto-placement dan Implicit Grids

Auto-placement:

Item tanpa grid-column atau grid-row akan ditempatkan secara otomatis dalam urutan dokumen.

Implicit Grid:

Ketika item ditempatkan di luar grid eksplisit, CSS Grid membuat baris/kolom implisit secara otomatis.

Properti terkait:

  • grid-auto-flow: Mengontrol arah auto-placement (row/column)
  • grid-auto-rows: Ukuran baris implisit
  • grid-auto-columns: Ukuran kolom implisit

Contoh: grid-auto-rows: 100px; membuat semua baris implisit berukuran 100px.
