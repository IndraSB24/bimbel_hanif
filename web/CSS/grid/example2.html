Demo CSS Grid: Grid Template Areas

grid-template-areas memungkinkan kita memberi nama pada area grid dan mengatur layout dengan cara yang lebih intuitif. Kita dapat mendefinisikan layout seperti ASCII art!

Penjelasan Grid Template Areas

Dalam CSS, kita mendefinisikan area seperti ini:

grid-template-areas:
    "header header header"
    "nav content sidebar"
    "footer footer footer";

Setiap baris mewakili baris grid, dan setiap kata mewakili area grid. Titik (.) dapat digunakan untuk area kosong.

Keuntungan:

  • Layout lebih mudah dibaca dan dipahami
  • Mudah mengubah posisi area tanpa mengubah HTML
  • Visual representation yang jelas
