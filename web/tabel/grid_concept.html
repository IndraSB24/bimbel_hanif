Grid Concept in HTML Tables

HTML tables are based on a grid system: rows and columns form a rectangular grid of cells.

Basic Grid: 3 Rows x 3 Columns

Cell 1,1 Cell 1,2 Cell 1,3
Cell 2,1 Cell 2,2 Cell 2,3
Cell 3,1 Cell 3,2 Cell 3,3

Each is a row, each is a cell in that row.

Irregular Grid with Merging

Tall Cell Cell 1,2 Cell 1,3
Cell 2,2 Cell 2,3
Cell 3,1 Wide Cell

Merging cells changes the grid structure but maintains the concept.
