HTML Table Attributes

Tables have various attributes to control appearance and behavior.

Attributes
  • border: Sets border width (deprecated, use CSS).
  • cellpadding: Space inside cells (deprecated, use CSS).
  • cellspacing: Space between cells (deprecated, use CSS).
  • width: Table width (deprecated, use CSS).
  • height: Table height (deprecated, use CSS).
  • align: Table alignment (deprecated, use CSS).
  • bgcolor: Background color (deprecated, use CSS).
  • summary: Summary for screen readers.

Attributes

  • align: Horizontal alignment (left, center, right).
  • valign: Vertical alignment (top, middle, bottom).
  • bgcolor: Row background color.
