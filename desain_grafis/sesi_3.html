2. Membuat Komponen UI (User Interface)

Di Canva, kita tidak perlu koding untuk membuat elemen UI. Kita menggunakan "Bentuk" (Shapes) dasar yang ada di menu Elemen.

  • Tombol (Button) Gunakan kotak (R), ubah warnanya, tumpulkan sudutnya (Corner Rounding), dan tambahkan teks di tengah.
  • Ketik di sini...
    Input Text / Form Gunakan kotak kosong, beri garis tepi (border), dan tambahkan teks abu-abu pudar di dalamnya (Placeholder).
  • 👤
    Avatar User Gunakan bingkai lingkaran (Frames) atau elemen lingkaran biasa (C).

3. Trik Ajaib: Menghubungkan Halaman (Linking)

🔗

Bagaimana membuat tombol berfungsi?

Inilah rahasia membuat Prototype di Canva! Kamu bisa menautkan (link) sebuah tombol atau teks agar saat diklik saat mode presentasi, layar akan otomatis berpindah ke halaman/slide tertentu.

Langkah-langkahnya:

  1. Pastikan kamu sudah punya minimal 2 Halaman desain (Misal: Hal 1 Login, Hal 2 Dashboard).
  2. Pilih elemen yang akan dijadikan tombol (Misal kotak biru bertuliskan "Masuk").
  3. Klik Kanan pada elemen tersebut, lalu pilih Tautan (Link). Atau tekan Ctrl + K.
  4. Pada kotak pencarian yang muncul, scroll ke bawah dan temukan daftar halamanmu.
  5. Pilih Halaman 2 (Dashboard). Klik Selesai.
  6. Klik tombol Tampilkan (Present) di pojok kanan atas Canva. Coba klik tombolmu!
