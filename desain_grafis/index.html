Kenapa Anak IT Harus Belajar Canva?

Di dunia perkuliahan dan industri, kode yang bagus seringkali tidak dilirik jika presentasinya buruk. Programmer masa kini dituntut bisa membuat Pitch Deck (Presentasi Proyek) yang meyakinkan, merancang UI Mockup cepat untuk presentasi ide, dan membangun Portofolio Visual yang menjual skill mereka.

  • Aturan kombinasi Font (Tipografi) yang modern.

Presentasi

Desain Pitch Deck (Presentasi Proyek)

  • Mengubah teks panjang menjadi poin visual (Infografis).

Tugas Praktik:

UI / UX

Prototyping & UI Mockup di Canva

Konten Edukasi

Tech Microblogging (Carousel Design)

  • Strategi membagikan ilmu IT di media sosial (Instagram/LinkedIn).

Tugas Praktik:
